# Publice

public → Members can be accessed from anywhere

# Private

private → Members can be accessed only inside the class

# Protected

protected members are accessible inside the same class and in derived (child) classes, but not accessible from outside using objects

# Encapsulation

Wrapping up of (data or properties) and member funtion in a single unite called class.

- It help in data hiding like puting data in private so it cannot be accessed but outside of class.
