# File mode in python 
r : read (default) — file must exist.
w : write — creates or truncates file.
x : exclusive creation — fail if file exists.
a : append — write at end, creates if missing.
b : binary mode — use with other flags (rb, wb).
t : text mode (default) — use with other flags (rt, wt).
+ : update (read/write) — combine (r+, w+, a+).
U : universal newlines — deprecated in Python 3 (avoid).
